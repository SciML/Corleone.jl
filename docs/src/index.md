

## Installation

To install `Corleone.jl`, run the following 

```julia
using Pkg
Pkg.add("Corleone")
```

## Optimal Experimental Design 

To derive optimal experimental designs [sager_2013_jan_samplingdecisionsoptimum](@cite) simply add `CorleoneOED.jl` by running 

```julia
using Pkg
Pkg.add("CorleoneOED")
```
