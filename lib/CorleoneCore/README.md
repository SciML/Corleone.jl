# CorleoneCore

Defines the core functionality for Corleone.jl, e.g. the computational nodes for the dynamics, signal definitions, their corresponding constructors, and a single shooting problem.

To install this library please run

```julia
using Pkg
Pkg.add(url="https://kosinus.math.uni-magdeburg.de/mathopt/software/corleone/", subdir = "lib/CorleoneCore", rev="Develop")
```
